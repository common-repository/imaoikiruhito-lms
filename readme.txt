== Description ==
Imaoikiruhito LMSは、LMS(Learning Management System)のプラグインです。
現在のところ、日本国内での使用を前提としています。

= テーマファイル =
こちらの[テーマファイル](https://imaoikiruhito-plugin.com/imaoikiruhito-lms-theme/imaoikiruhito_lms.zip)をお使いください。

= 日本製のLMS =
日本製なので、翻訳がわかりづらい…ということがありません。

= カード決済に対応 =
PayPalを使った決済が可能です。

= プラグイン紹介ページ =
プラグインの[公式サイト](https://imaoikiruhito-plugin.com/)

### 使い方
- 「PayPal設定」にて決済に必要な情報を登録します。
- 「レッスン」にて動画を登録します。
- 「コース」にてレッスンとコースを紐付けします。
- 「講座」にて講座とコースを紐付けします。
- ユーザーは「講座」を購入して受講します。

== Installation ==

= 自動インストール =

1.ワードプレス管理画面のプラグイン検索(プラグイン > 新規追加)にアクセス
2.「Imaoikiruhito LMS」と入力し、「プラグインの検索」をクリック
3.プラグインが見つかったら「今すぐインストール」をクリック
4.インストール後、プラグインを有効化

= 手動インストール =

1.プラグインをダウンロード
2.imaoikiruhito-lms.zipフォルダを/wp-content/plugins/ディレクトリにアップロード
3.プラグイン > インストール済みプラグインでプラグインを有効化
